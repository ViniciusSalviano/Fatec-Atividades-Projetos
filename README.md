## 📂 Estrutura do Repositório

Seguindo o padrão de organização do ambiente Java no VS Code:

*   **`src/`**: Contém todos os códigos-fonte originais (`.java`)[cite: 1].
*   **`lib/`**: Reservada para bibliotecas e dependências externas[cite: 1].
*   **`bin/`**: Onde os arquivos compilados (`.class`) são gerados automaticamente[cite: 1].
